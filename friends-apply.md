## 什么是友情链接？

友情链接（Friend Links）是博客之间的互相推荐。通过交换友链，我们可以结识志同道合的博主，拓展阅读视野，构建独立博客的互联网络。

在这个社交媒体平台化的时代，友链保持着互联网早期的开放精神——不依赖算法推荐，而是通过博主之间的信任与欣赏，主动建立连接。

## 如何申请友链？

### 📋 友链信息格式

请按以下格式提供您的博客信息：

```yaml
- name: "你的博客名称"
  url: "https://your-blog.com"
  description: "一句话介绍你的博客（建议 20-50 字）"
  imageUrl: "https://your-blog.com/avatar.png"
```

### 字段说明

| 字段 | 说明 | 是否必填 | 要求 |
|------|------|----------|------|
| **name** | 博客名称或昵称 | ✅ 必填 | 建议不超过 15 字 |
| **url** | 博客主页链接 | ✅ 必填 | 应包含 `https://`，确保可正常访问 |
| **description** | 博客简介 | ✅ 必填 | 一句话概括博客内容/特色，50 字以内为佳 |
| **imageUrl** | 头像/Logo 图片 | ⭕ 选填 | 建议正方形图片，200×200px 以上 |

### 提交方式

**方式一：提交 Issue（推荐）**

1. 前往 [GitHub Issues](https://github.com/Lithiumcr/Lithiumcr.github.io/issues/new)
2. 标题格式：`[友链申请] 你的博客名称`
3. 正文按上述 YAML 格式填写友链信息

**方式二：提交 Pull Request**

1. Fork 本仓库
2. 编辑 `_data/friends.yml` 文件，在末尾添加您的友链信息
3. 提交 PR，标题：`Add friend link: 你的博客名称`

## 友链要求

为了保证友链质量和相关性，请确保您的博客符合以下条件：

*   ✅ **原创内容**：以原创内容为主的个人博客
*   ✅ **正常访问**：网站可正常访问，无频繁宕机
*   ✅ **相互友链**：希望您也能在博客中添加本站友链（非强制）

### 优先接受的博客类型

*   🤔 人文、历史、社科相关
*   🎮 游戏、技术、文化相关
*   📝 个人随笔、生活记录

## 常见问题

**Q: 必须互换友链吗？**
A: 不强制，但互换友链的申请会优先考虑。

**Q: 我的博客刚建立，文章不多可以吗？**
A: 可以！内容原创、有趣即可，不看文章数量。

**Q: 图片 URL 从哪里获取？**
A: 可以使用您博客的头像、Logo 或任何代表性图片。如果没有，可以暂时不提供 `imageUrl` 字段。

**Q: 友链会一直保留吗？**
A: 如果您的博客长期不更新或无法访问，可能会被移除。移除前会尝试联系。

**Q: 可以修改已提交的友链信息吗？**
A: 当然可以！提交新的 Issue 或 PR 说明修改内容即可。
